# Getting Started

### Prerequisites
Make sure you have the following installed:

- **Java**: JDK 17+ (recommended)
- **Maven**: 3.9+
- **Git**

### Accessing the application

http://localhost:8080

### Unit test

mvn test

### Logging

Logs are configured using log4j2.xml

### Swagger UI

http://localhost:8080/swagger-ui/index.html

### OpenAPI JSON

http://localhost:8080/v3/api-docs
